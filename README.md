# ERP-Project-frontend
Gladiator project - ERP - Project Manangement tool - frontend

Includes the following angular components:
User Login - 
  -Manager Login
  -Associate Login
Project Directory - 
  - Mananger Access
        - Add Project
        - Add comments, proposals and tasks to Employees
        - check and reply to employee feedback
        - tag associates to a project
  - Associate Access
        - add comment to projects and request proposals to managers
        - add task feedbacks
        - add rating to Managers and tasks
  - Graph based analytics on projects, tasks and Manager/Employee feedbacks and rating using Apex Chats
